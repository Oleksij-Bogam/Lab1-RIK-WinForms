# Lab1 — Survey Application (Windows Forms)

Лабораторна робота №1  
Дисципліна: РІК  
Тема: Створення графічного застосунку з використанням **Windows Forms**

## Опис проєкту

Цей застосунок реалізує просту **форму опитування користувача**.  
Користувач вводить відповіді на запитання у текстові поля, після чого результати зберігаються у файл на диску.

Метою лабораторної роботи є ознайомлення з базовими можливостями **Windows Forms** та створення простого GUI-застосунку на **C#**.

## Функціональність

Застосунок дозволяє:

- відображати форму з питаннями
- отримувати відповіді користувача через текстові поля
- зберігати відповіді у файл
- використовувати кнопки для керування процесом опитування
- працювати з подіями інтерфейсу (Button Click)

## Технології

- C#
- .NET
- Windows Forms
- Visual Studio

## Приклад роботи

1. Користувач запускає програму.
2. З'являється форма опитування.
3. Користувач вводить відповіді на питання.
4. Після натискання кнопки відповіді зберігаються у файл.

## Результат

Програма створює файл з відповідями користувача, який зберігається на диску.
